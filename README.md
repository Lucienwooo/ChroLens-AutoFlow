# AutoFlow 操作實務手冊

**AutoFlow** (ChroLens 系列) - 是一款高效的影片自動分類與多窗播放工具。本手冊說明如何使用本程式的所有核心功能。

---

## 🚀 核心功能操作說明

### 1. 影片自動分類 (AutoFlow)

- **選取路徑**：點擊左下角「瀏覽...」按鈕，選擇存放影片的目標資料夾。
- **智能分析**：點擊「開始」按鈕，程式將自動辨識影片編號，並從數據庫抓取對應的女優名稱。
- **自動分類**：抓取成功後，影片將自動移動至依女優名稱命名的子資料夾中，實現井然有序的庫存管理。

### 2. 影片列表互動

- **即時預覽**：滑鼠懸停於清單中的影片項目上，左右移動滑鼠即可在縮圖區即時預覽影片不同時間點的畫面。
- **播放/暫停**：點擊列表中的影片畫面或「播放」按鈕，即可開啟內嵌播放器。再次點擊畫面可切換播放/暫停。
- **快速操作**：每個項目配備獨立的「更名」與「刪除」按鈕，方便您在整理過程中即時修改檔名或清理檔案。
- **音量控制**：每個清單項目都有獨立的音量滑桿，可個別調整內嵌播放的音量。

### 3. 多窗瀏覽器 (9宮格播放)

- **開啟方式**：點擊主面板底部的「多窗瀏覽」按鈕開啟獨立播放視窗。
- **拖放播放**：從左側影片清單將檔案拖曳至右側 9 個播放器視窗中的任意一個即可開始播放。
- **覆蓋功能**：若視窗已有影片，直接將新影片拖入該視窗即可直接替換內容。
- **右鍵清除**：在播放視窗上點擊「滑鼠右鍵」，可直接清除該視窗載入的影片內容。
- **暫停/播放**：左鍵點擊播放畫面即可快速切換播放或暫停狀態。

### 4. 系統更新與關於

- **版本檢查**：點擊「關於」按鈕可開啟版本資訊視窗。
- **一鍵更新**：視窗會自動檢查有無新版本，若有更新可用，點擊「立即更新」即可自動完成下載與安裝。
